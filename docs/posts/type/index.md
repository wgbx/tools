# Type

## isNumber

🎯判断给定的值是否为数值类型

```typescript
/**
 * @param {*} val - 需要进行类型判断的任意值
 * @returns {boolean} - 如果传入值是数值类型，则返回 `true`，否则返回 `false`
 * @example
 * console.log(isNumber(123)); // 输出 `true`
 * console.log(isNumber('hello')); // 输出 `false`
 */
```

## isString

 * @example

 * console.log(isString(123)); // 输出 `false`
 */
